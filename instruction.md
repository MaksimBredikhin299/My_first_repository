# Инструкция по языку MarkDown

## Стилизация текста
Чтобы выделить текст курсивом обрамите его звездочками

*Курсив.*

Чтобы выделить текст полужирным обрамите его двумя звездочками

**Полужирный**

Чтобы сделать список поставьте звездочку перед пунктом
### Список
* Элемент 1
* Элемент 2
* Элекмент 3

1. элемент
2. элемент
